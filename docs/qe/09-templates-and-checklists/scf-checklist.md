# SCF Checklist

- [ ] 结构来源明确。
- [ ] 晶胞、坐标、元素数、真空层已检查。
- [ ] 赝势来源、泛函、类型一致。
- [ ] `ecutwfc` 与 `ecutrho` 有选择依据。
- [ ] k 点设置与维度、金属性匹配。
- [ ] occupation/smearing 设置合理。
- [ ] `prefix/outdir` 与下游计划一致。
- [ ] output 显示 SCF 收敛。
- [ ] 记录 total energy、Fermi energy、SCF accuracy、warnings。
- [ ] 判断为 PASS/WARN/BLOCK。
