# Common QE errors

## 文档目的

收集常见 QE 错误信息，并把错误映射到 workflow 阶段、可能原因和修复顺序。

## 待收集条目

- input namelist/card 语法错误。
- pseudopotential 文件找不到或格式不兼容。
- SCF not converged。
- ph.x dyn 文件不完整。
- outdir/prefix 不匹配。
- MPI/SLURM 资源错误。

## 记录格式

```markdown
## Error message

## Workflow stage

## Likely cause

## Evidence

## Fix order

## Preventive checklist
```
