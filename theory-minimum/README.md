# Theory Minimum

本目录只记录正确使用 Quantum ESPRESSO 所需的最低理论背景。每篇理论页都应落到 QE input 字段、output 审阅位置和 workflow 准入判断。

## 阅读方式

先从 `learn/` 的能力路线进入具体 workflow；遇到参数或 output 判断无法解释时，再回到本目录查对应理论页。
