## 使用规则

- 官方 input reference 用于核对参数定义和适用条件。
- Pranab / Kyoto 等教程用于学习工作流拆解和 output 判断。
- ASE、pymatgen、SeeK-path、phonopy、Wannier90 等只作为辅助参考，不替代 QE 原生命令行学习。
- AiiDA / AiiDA-QE 只作为高级 workflow/provenance 参考。
