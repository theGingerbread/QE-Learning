# Phonopy QE Interface

## 它适合解决什么学习问题？

了解有限位移 phonon 与 QE 的接口，作为 DFPT phonon 的对照和高级参考。

## 覆盖哪些 QE workflow？

- finite-displacement phonon
- phonon band structure
- phonon DOS

## 适合跟读的部分

- 学习 Phonopy 如何读写 QE force 相关数据和组织位移超胞。

## 只适合作为参考的部分

- 本仓库 phonon 主线优先写 QE DFPT；Phonopy 接口作为补充。

## 局限

有限位移和 DFPT 的前提、误差来源和超胞要求不同，不能混用审阅标准。
