# VESTA / XCrySDen

## 1. 它是什么？

VESTA 和 XCrySDen 是 QE 生态中常用的结构、体数据、费米面和模式可视化工具。

## 2. 适合学习或解决什么问题？

适合检查结构、charge density、potential、phonon modes、Fermi surface。

## 3. 它覆盖哪些 QE workflow？

post-processing、structure inspection、phonon mode visualization、Fermi surface visualization。

## 4. 哪些部分适合跟做？

用 `pp.x` 输出的体数据或 `fs.x` 的 BXSF 文件做可视化。

## 5. 哪些部分只适合作为参考？

图像工具不判断结果是否收敛。

## 6. 它和本仓库哪些页面对应？

- `learn/07-postprocessing-loop.md`
- 结构操作独立学习项目

## 7. 它的局限是什么？

可视化容易制造直观错觉，必须保留原始数据、单位和生成脚本。
