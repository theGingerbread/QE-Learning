# ENCCS QE HPC Tutorials

## 它适合解决什么学习问题？

学习在 HPC 环境运行 QE、理解并行执行、作业脚本和性能相关边界。

## 覆盖哪些 QE workflow？

- SCF on HPC
- phonon on HPC
- parallel execution and job scheduling

## 适合跟读的部分

- 适合跟读 Slurm/job script 组织、MPI 运行方式和作业资源设置。

## 只适合作为参考的部分

- HPC 参数不替代物理收敛；资源设置和科学可信度需要分开记录。
