# Official QE Documentation

## 它适合解决什么学习问题？

核对 QE 程序、输入字段、默认行为和官方 package 边界。它是参数定义的主要来源。

## 覆盖哪些 QE workflow？

- PWscf / pw.x
- PHonon / ph.x
- PostProc
- PWneb
- CP
- XSPECTRA
- TDDFPT
- EPW
- atomic and related packages

## 适合跟读的部分

- 使用 `INPUT_*` 页面核对参数名称、类型和适用程序。
- 使用 package user guide 核对程序链和文件依赖。

## 只适合作为参考的部分

- 官方文档不替代 hands-on 学习；读者仍需结合 workflow 页面写 output review。
