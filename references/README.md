# References

`references/` 用于整理公开文档、教程网站和辅助工具的学习导读。参数定义仍以 QE 官方 `INPUT_*` reference 为准；工具页只说明它们在 QE workflow 中能解决什么问题和边界在哪里。

## 使用方式

- 查参数定义：优先读 [official-qe-docs.md](tutorial-sites/official-qe-docs.md)。
- 查 hands-on 拆解：读 [pranab-das-qe-tutorial.md](tutorial-sites/pranab-das-qe-tutorial.md)。
- 查 phonon 判断：读 [kyoto-phonon-dokuwiki.md](tutorial-sites/kyoto-phonon-dokuwiki.md)。
- 查工具边界：读 `tools/` 下对应页面。
